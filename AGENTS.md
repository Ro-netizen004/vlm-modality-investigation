# Agent instructions

AI coding agents working in this repository must read **`CLAUDE.md`** first.

That file defines:

- Canonical pipeline (`src/` + notebooks, not duplicate frameworks)
- Image dataset v1 vs v2 (do not mix)
- Files you must not replace blindly
- Git and reproducibility rules

For architecture details see **`docs/CANONICAL.md`**.

**Before any PR:** `git pull origin main` · extend existing modules · update `CLAUDE.md` if behavior changes.
